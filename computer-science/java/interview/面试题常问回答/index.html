面试题常问回答

  1. 上份工作离职理由

    由于公司组织结构调整，需要从我所在原来的部门调到其他岗位，同事有接受调岗，也有在观望，但我不想偏离自己的职业方向，所以只好离职。

  2. 空窗期在干什么？或者空窗期找工作的情况？

    上份工作离职后，在找工作的过程中，发现我在开发技术方面很多地方需要提升，所以自学了一些符合就业市场中需求的技术课程，趁此机会提升自己的能力，以便能够更好地胜任工作。

  3. 当前你的找工作情况，目前手上有offer吗？

    说实话，我最近才开始陆续参加一些面试，有几个公司已经到了最后环节，offer还在洽谈中。
    我自己也是偶然看到贵公司的岗位，觉得自己与该岗位十分合适，另外过往的工作经历和贵岗位非常匹配，也很喜欢这个平台和氛围，尤其是今天和您聊了之后，对公司XX方面很佩服，真心期待有这样一个机会。

  4. 你能接受加班吗？

    我会努力地避免因为个人低效等原因造成的加班问题。

  5. 自我介绍一下？

    面试官您好，我叫罗涛，从2022年开始进行java开发的工作，到目前为止，已经有了1年左右的工作经验，我在一家公司工作过，参与过一些应用的二次开发，比如说，机构服务平台。在工作期间呢，也做出来了一些成绩，比如说在机构服务平台这个系统中，我为公司重构了登录和安全认证模块相关的功能，包含了用户管理、权限管理、角色管理、验证码认证、密码加密等等功能，为公司将老系统基础模块二次开发后，提供了更加方便且易扩展的程序。用到的技术栈主要以SpringBoot为主，对于其他的技术也有一定的了解，如Redis、SpringSecurity、Mybatis。好了，我的介绍完毕。

  6. 你还有什么问题需要问我的吗？

    我想请问一下，如果确认录用的话，我需要准备什么可以快速融入到贵司的工作当中？

    刚才的面试当中，你觉得我的技术存在什么问题？下一步应该怎么学习？

    你觉得我刚才面试当中哪个方面回答的不好？

    咱们公司的技术栈是什么？

    咱们公司内部有没有什么培训或者分享

  7. 如果录用你的话，需要多久能入职？

    如果贵公司录用了我的话，我会在一周内给出准确的答复。

  8. 介绍一下你的缺点？

    由于我是刚毕业出来1年多，所以我的社会经验不足，可能在一些场景下沟通技巧和职场经验上不足，但是我会努力学习，尝试改变这种境况。
